# IMAP Backup Application

A fullstack web application for backing up IMAP email accounts.

## Project Structure

- `backend/` - Node.js/Express server
- `frontend/` - React/Vite application
- `data/` - Local storage for email backups

## Getting Started

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features

- Add and manage IMAP email accounts
- Download and backup all folders and messages
- Export emails to another account
- Modern UI with TailwindCSS
